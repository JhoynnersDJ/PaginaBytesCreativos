Texto creativo en fondo azul

¡Hola! Este es un ejemplo de un div con un fondo creativo en tonos de azul. El fondo utiliza un degradado que combina diferentes tonalidades de azul, con el color base #0397d6ea, para crear un efecto visualmente atractivo. El texto dentro del div está centrado vertical y horizontalmente, y se muestra en color blanco sobre el fondo azul. ¡Espero que te guste!
